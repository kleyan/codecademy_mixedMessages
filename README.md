# codecademy_mixedMessages
Repository for the Codecademy Portfolio Project "Mixed Messages" in the Full-Stack Engineer Career Path

##Technologies
* JavaScript
* NodeJS

##How it works:
It uses three arrays consisting of sentence pieces as a starting point. The following function randomly selects
one item of the parameter-array as an item for the sentence. The second function returns the formated sentence by
calling the first function for every array.


###How to change the sentences
You can change the sentence options by changing the arrays from the start of the program.
